## 🧪 How to Run Locally

1. Clone the repo:
    ```bash
    git clone https://github.com/your-username/ai-war-dashboard.git
    cd ai-war-dashboard
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the app:
    ```bash
    streamlit run aiwarapp.py
    ```
